# Organization Context

## Purpose

Provides background information about the client organization to inform
all SDLC activities. Customize per engagement.

## Usage

1. Copy [org-profile.template.md](org-profile.template.md) for new engagements
2. Fill in during Phase 0 discovery
3. Update as you learn more
4. Reference throughout all phases

## Contents

The organization profile includes:

### Company Overview
- Industry and market position
- Size and structure
- Key business drivers
- Strategic priorities

### Stakeholder Map
- Decision makers and sponsors
- Influencers
- End users
- Technical teams

### Technical Landscape
- Current systems
- Technology preferences
- Standards and constraints
- Integration landscape

### Project History
- Prior initiatives
- Past successes and failures
- Organizational trauma points
- Lessons from history

## When to Load

- Start of Phase 0 (required)
- When onboarding new team members
- When making decisions affected by org context
- When presenting to stakeholders
